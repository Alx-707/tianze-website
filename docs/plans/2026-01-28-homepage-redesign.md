# 首页内容重构设计

## 背景

当前首页内容未围绕核心卖点组织。需要重构以突出天泽管业的差异化定位：**设备 + 管件一体化制造商**。

## 公司发展历程（叙事支撑）

```
2018-2021: 专注弯管设备研发制造（半自动/全自动弯管机）
    ↓
2021至今: 设备 + 管件 双轮驱动
```

**这是核心差异化的来源**——先做设备，再做管件，技术能力自下而上生长。

## 核心竞争力（5项）

| 优先级 | 竞争力 | 首页体现 |
|--------|--------|----------|
| ★★★ | 弯管制造系统性能力（设备→模具→工艺→管件） | 技术根基区块 |
| ★★★ | 设备研发制造能力 | 技术根基区块 |
| ★★ | 非标定制 + 快速打样 | 产品矩阵-定制方案 |
| ★★ | 气动物流管服务能力 | 产品矩阵-气动管 |
| ★ | 一体化制造体系 | 数据条/关于页 |

## 核心卖点

- 自研弯管设备（不是贸易商）
- 设备 + 管件双业务线，互相背书
- 全自动生产，质量稳定，交期可控
- 服务国内一线管道品牌代加工（隐喻，不点名）

## 目标客户关心的点

| 关心点 | 回应方式 |
|--------|----------|
| 性价比 | 工厂直供 |
| 成品标准 | 自研设备，精度可控 |
| 原料质量 | 100% 全新原料 |
| 交期可靠 | 全自动生产 |
| 供应商可信 | 60% 复购率 |

## 首页整体结构

```
Hero: 我是谁（一句话差异化）
  ↓
产品矩阵: 弯管设备 | PVC弯管 | 气动物流管 | 定制方案（4列）
  ↓
CTA: 获取报价
```

## 区块设计

### 1. Hero 区块

**目标**: 10 秒内让访客明白"天泽是设备+管件一体化制造商，不是贸易商"

**布局**: 居中布局，文字在上，设备图组在下

**内容**:

| 元素 | 内容 |
|------|------|
| 标签 | "关于我们 →"（可点击跳转） |
| 主标题 | 设备 + 管件<br>一体化制造商 |
| 副标题 | 自研弯管设备，自产精密管件。<br>从设备到成品，全链路可控。 |
| 设备图组 | 2-3 张图片并排（弯管机、扩管机、生产线），移动端横向滚动 |
| 数据条 | 工厂直供 · 100% 新料 · 全自动生产 · 60% 复购率 |
| 引导 | 居中"了解更多" + 向下箭头（浮动动画）→ 滚动到产品矩阵 |

**设计决策**:

1. **去掉 Hero 区域的"获取报价"按钮**
   - 用户刚进入页面，尚未了解信息，不会直接询价
   - 导航栏始终有"获取报价"按钮，无需重复
   - Hero 应引导用户继续浏览，而非打断

2. **数据条从客户视角设计**
   - 原数据（出口国家、年经验）是自我视角
   - 新数据回应客户实际关心的问题

3. **去掉 ISO 9001 标签**
   - 太普遍，不是差异化卖点
   - 改为"关于我们"入口

**设备图组**:
- 2-3 张设备照片并排展示
- 桌面端：3 列网格
- 移动端：横向滚动（snap scrolling）
- 图片比例：4:3 横向
- 首选内容：弯管机、扩管机、生产线

**目标**: 展示四条业务线，弯管设备作为产品之一（同时体现技术实力）

**布局**:

│  │  弯管设备    │   PVC弯管    │  气动物流管  │   定制方案   │     │
│  │              │              │              │              │     │
│  │  [图片]      │  [图片]      │  [图片]      │  [图片]      │     │
│  │              │              │              │              │     │
│  │  自主设计    │  喇叭口弯管  │  PVC/PMMA/   │  低MOQ       │     │
│  │  研发制造    │  双承口弯管  │  PETG        │  快速打样    │     │
│  │              │              │              │              │     │
│  │  半自动/     │  UL651/AS/   │  高透明静音  │  非标定制    │     │

│  │              │              │              │              │     │
│  │  [了解更多]  │  [了解更多]  │  [了解更多]  │  [获取报价]  │     │
│  └──────────────┴──────────────┴──────────────┴──────────────┘     │

**设计决策**:
- 弯管设备放第一位，"自主设计研发制造"体现技术实力
- 不需要单独的"技术根基"区块，设备产品本身就是技术实力证明
- 四列平铺，移动端自动响应为 2x2 或单列

**各产品线内容**:

| 产品线 | 核心卖点 | 具体产品 | 标准/场景 |
|--------|----------|----------|-----------|
| 弯管设备 | 自主设计研发制造 | 半自动/全自动弯管机、扩管机 | 服务国内一线品牌 |
| PVC弯管 | 符合国际标准 | 喇叭口、双承口、大弧度、弯头、扩口件 | UL651/AS/NZS/GB |
| 气动物流管 | 高透明静音防漏 | PVC/PMMA/PETG透明管 | 医院/实验室/工业物流 |
| 定制方案 | 低MOQ快速打样 | 非标规格、特殊弯度、OEM | 澳标/美标/国标 |

### 3. CTA 区块

**状态**: 已确认

**布局**: 极简单行，全宽渐变背景

**内容**:

| 元素 | 内容 |
|------|------|
| 文案 | "联系我们，获取样品或技术支持" |
| 按钮 | "获取报价"（实心白底蓝字，整页唯一强调按钮） |

**设计决策**:
- 务实导向文案，覆盖两种常见询盘场景（样品 + 技术支持）
- 渐变背景（主色 → 深主色），与页面其他区块形成视觉区分
- 整页只有这一个实心按钮，其他按钮均为描边样式

## UI 设计规范

### 颜色

| 用途 | 规范 |
|------|------|
| 主色 | 工业蓝（已有 CSS 变量 `--primary`） |
| 背景 | 白/浅灰 |
| 文字 | 深灰/黑 |
| CTA 渐变 | 主色 → 深主色 |

### 按钮层级

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主要 | 实心蓝底白字 | **仅** CTA 区块的"获取报价" |
| 次要 | 描边蓝框蓝字 | 产品卡片、Hero 引导、其他所有按钮 |

### 圆角

| 元素 | 圆角 |
|------|------|
| 卡片 | 12px |
| 按钮 | 8px（shadcn 默认） |
| 图片 | 与卡片一致 12px |

### 间距

| 类型 | 数值 |
|------|------|
| 区块间距 | 80-120px |
| 区块内间距 | 40-60px |
| 卡片间距 | 24px |

### 响应式布局

| 区块 | 桌面 | 移动端 |
|------|------|--------|
| Hero 设备图 | 3 列网格 | 横向滚动 |
| 产品矩阵 | 4 列 | 2x2 网格 |
| CTA | 单行 | 单行（文字+按钮堆叠） |

### 产品卡片

- **信息密度**: 详细型（图片 + 标题 + 2-3 要点列表 + 按钮）
- **图片比例**: 4:3 横向
- **悬停效果**: 轻微上浮 + 阴影加深
