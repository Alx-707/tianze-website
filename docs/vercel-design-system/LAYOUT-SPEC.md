# Vercel 风格布局规范

> 从 Vercel 官网提取的布局设计规范，可直接迁移到其他项目使用

---

## 1. 容器布局 (Container)

### 核心概念

容器用于约束内容的最大宽度，并使其在页面中水平居中。

```
浏览器宽度: 1440px
┌──────────────────────────────────────────────────────┐
│                                                      │
│  180px  ┌─────── 内容区 1080px ───────┐  180px      │
│  (自动) │                              │  (自动)     │
│         │     max-width: 1080px        │             │
│         │     margin: 0 auto           │             │
│         └──────────────────────────────┘             │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 设计数值

| 属性 | 值 | 说明 |
|------|-----|------|
| 内容最大宽度 | `1080px` | 主要内容区域 |
| 窄内容宽度 | `860px` | 文章、表单等 |
| 水平居中 | `margin: 0 auto` | 左右自动 |
| 边距保护 | `padding: 0 24px` | 小屏幕时的最小边距 |

### 容器变体

```
┌─────────────────────────────────────────┐
│  max-width: 1280px  - 全宽仪表盘        │
├─────────────────────────────────────────┤
│  max-width: 1080px  - 标准页面 (默认)    │
├─────────────────────────────────────────┤
│  max-width: 860px   - 窄内容 (文章)      │
├─────────────────────────────────────────┤
│  max-width: 640px   - 超窄 (登录表单)    │
└─────────────────────────────────────────┘
```

## 2. 响应式断点 (Breakpoints)

### 核心断点

| 断点名称 | 宽度 | 触发场景 | 典型设备 |
|---------|------|---------|---------|
| `xs` | < 375px | 超小屏手机 | iPhone SE |
| `sm` | ≥ 640px | 大屏手机 | iPhone Pro Max |
| `md` | ≥ 768px | 平板竖屏 | iPad |
| `lg` | ≥ 1024px | 平板横屏/小桌面 | iPad Pro |
| `xl` | ≥ 1150px | **导航切换点** | 笔记本 |
| `2xl` | ≥ 1400px | 大桌面 | 外接显示器 |

### 关键断点行为

```
1150px - 导航栏切换
─────────────────────────────────────────
> 1150px  │  桌面导航：完整链接展开
≤ 1150px  │  移动导航：汉堡菜单 + 抽屉

960px - 内容布局调整
─────────────────────────────────────────
> 960px   │  多列布局
≤ 960px   │  堆叠布局

768px - 平板适配
─────────────────────────────────────────
> 768px   │  双列卡片
≤ 768px   │  单列卡片

640px - 移动端优化
─────────────────────────────────────────
> 640px   │  标准间距
≤ 640px   │  紧凑间距
```

### 容器在不同断点的表现

| 断点 | 容器宽度 | 左右边距 |
|------|---------|---------|
| < 640px | 100% | 16px (固定) |
| 640px - 1080px | 100% | 24px (固定) |
| 1080px - 1440px | 1080px | 自动居中 |
| > 1440px | 1080px | 自动居中 (更大留白) |

## 3. 装饰性网格线 (Decorative Grid)

### 设计意图

- 增加科技感/工程感
- 暗示精确和专业
- 作为内容的视觉锚点
- 强化品牌识别

### 设计数值

| 属性 | 浅色模式 | 深色模式 |
|------|---------|---------|
| 线条颜色 | `rgba(0, 0, 0, 0.05)` | `rgba(255, 255, 255, 0.05)` |
| 线条粗细 | `1px` | `1px` |
| 网格单元 | `~90px × 90px` | `~90px × 90px` |
| 交叉标记 | `11×21px` 或 `21×11px` | 同左 |

### 视觉层级

```
z-index 层级:
─────────────────────────────────────────
z-index: 0   │  装饰网格线 (最底层)
z-index: 1   │  背景元素
z-index: 10  │  主要内容
z-index: 50  │  导航/Header
z-index: 100 │  模态框/弹窗
```

### 关键特性

1. **极度克制** - 只有 5% 不透明度
2. **不干扰内容** - `pointer-events: none`
3. **装饰性标记** - `aria-hidden="true"`
4. **性能友好** - 纯 CSS 实现，无 JS

## 4. 间距系统 (Spacing)

### 8px 基准网格

所有间距值应为 8 的倍数：

```
4px   - 半步 (例外情况)
8px   - 1x  基准
16px  - 2x  常用
24px  - 3x  常用
32px  - 4x
40px  - 5x
48px  - 6x
64px  - 8x
80px  - 10x
96px  - 12x
```

### 应用场景

| 间距 | Tailwind | 使用场景 |
|------|----------|---------|
| 8px | `gap-2`, `p-2` | 图标与文字、紧凑元素 |
| 16px | `gap-4`, `p-4` | 卡片内边距、列表项间距 |
| 24px | `gap-6`, `p-6` | 区块内边距、表单字段间距 |
| 32px | `gap-8`, `p-8` | 模块间距 |
| 48px | `gap-12`, `py-12` | 大区块分隔 |
| 64px | `gap-16`, `py-16` | 页面区块分隔 |

## 5. 快速参考

### CSS 变量定义

```css
:root {
  /* 容器 */
  --container-max: 1080px;
  --container-narrow: 860px;
  --container-padding: 24px;
  
  /* 断点 */
  --breakpoint-sm: 640px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1150px;
  --breakpoint-2xl: 1400px;
  
  /* 网格线 */
  --grid-line-color: rgba(0, 0, 0, 0.05);
  --grid-line-width: 1px;
  --grid-cell-size: 90px;
  
  /* 间距 */
  --space-1: 4px;
  --space-2: 8px;
  --space-3: 12px;
  --space-4: 16px;
  --space-6: 24px;
  --space-8: 32px;
  --space-12: 48px;
  --space-16: 64px;
}

/* 深色模式 */
[data-theme="dark"] {
  --grid-line-color: rgba(255, 255, 255, 0.05);
}
```

### Tailwind 配置概览

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    screens: {
      'sm': '640px',
      'md': '768px',
      'lg': '1024px',
      'xl': '1150px',   // 导航切换点
      '2xl': '1400px',
    },
    extend: {
      maxWidth: {
        'container': '1080px',
        'narrow': '860px',
      },
    },
  },
}
```
