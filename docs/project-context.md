# Project Context

这份文档只放项目背景、公司信息、业务信息，主要给 AI 做内容生成、设计生成时读取。

## 项目定位

Tianze Website 是天泽管业的海外站点项目，主目标是把网站做成一个面向海外买家的询盘转化入口，而不是单纯的品牌展示页。

当前最重要的结果是：

- 让目标买家理解 Tianze 卖什么
- 让买家快速建立信任
- 把流量导向 Contact 询盘主路径

## 公司与品牌

- 公司：连云港天泽管业有限公司 / Lianyungang Tianze Pipe Industry Co., Ltd.
- 品牌：Tianze / 天泽
- 类型：Manufacturer + Trader（工贸一体）
- 运营起始：2018 年起做相关业务，2023 年正式注册
- 规模：约 60 人，技术团队约 10 人，厂区约 100 亩

## 业务核心

Tianze 的核心差异化不是“普通管材卖家”，而是：

> **Not just a pipe seller — factory-controlled forming, tooling, and OEM manufacturing partner.**

也就是：

- 内部掌握关键成型工艺
- 自己做模具和打样迭代
- 用可控的生产流程制造管材和配件

这直接带来几件事：

- 成型工艺理解更深
- 非标规格更灵活
- 打样和模具迭代更快
- OEM/定制能力更有说服力

## 产品线

当前内容与设计生成默认围绕四条产品线理解项目：

1. **PETG 气动传输管系统**
   - 高透明
   - 防漏
   - 静音
   - 面向医院、实验室、物流传输场景

2. **PVC 电工套管系统（美标 / 其他出口标准）**
   - Schedule 40/80
   - ASTM / UL651 / AS/NZS 等合规能力语境
   - 面向建筑电气、预埋、穿线

3. **PVC 给排水类产品**
   - 作为补充产品线展示

4. **内部成型、模具与 OEM 能力**
   - 这是制造过程能力，不是当前公开网站的独立产品线
   - 默认偏“信任背书 + 能力证明”表达
