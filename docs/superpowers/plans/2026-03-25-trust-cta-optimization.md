# Batch 2: Trust Enhancement + CTA Optimization — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Strengthen trust signals and inquiry conversion on the Tianze website by updating hero positioning, fixing certification display accuracy, and adding WhatsApp to the contact page.

**Architecture:** Translation-file-first approach. Most changes are JSON key updates in the split quartet (`messages/{en,zh}/critical.json`) with legacy flat files auto-regenerated by the pre-commit hook (`i18n-regenerate-flat`). One component refactor (QualitySection) replaces flat cert badges with a two-tier certified/compliant layout. Contact page gets a WhatsApp row via `getContactCopy.ts` helper update.

**Tech Stack:** Next.js 16 + next-intl 4.8 + Tailwind CSS 4.2 + Lucide React icons

**Spec:** `docs/superpowers/specs/2026-03-25-trust-cta-design.md`

Detect placeholder state:
```typescript
const WHATSAPP_PLACEHOLDER = "+86-518-0000-0000";
const whatsappNumber = siteFacts.contact.whatsapp;
const isWhatsAppConfigured = whatsappNumber !== undefined
  && whatsappNumber !== WHATSAPP_PLACEHOLDER;
const whatsAppUrl = isWhatsAppConfigured
  ? `https://wa.me/${whatsappNumber.replace(/[^0-9]/g, "")}`
  : undefined;
```

Add the row (same structure as email/phone rows):
```tsx
<div className="flex items-center space-x-3">
  <div className="flex h-10 w-10 items-center justify-center rounded-lg bg-primary/10">
    <MessageCircle className="h-5 w-5 text-primary" />
  </div>
  <div className="flex-1">
    <p className="font-medium">
      {copy.panel.whatsapp.label}
    </p>
    <p className="text-muted-foreground">
      {isWhatsAppConfigured ? whatsappNumber : copy.panel.whatsapp.comingSoon}
    </p>
  </div>
  {whatsAppUrl !== undefined && (
    <a
      href={whatsAppUrl}
      target="_blank"
      rel="noopener noreferrer"
      className="text-sm font-medium text-primary hover:underline"
    >
      {copy.panel.whatsapp.chatNow} →
    </a>
  )}
</div>
```
