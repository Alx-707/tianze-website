# 06-open-questions-and-missing-info

为了完善网站内容并确保信息准确，请补充以下关键信息：

## 1. 公司基础实力 (Company Profile)
- [ ] **成立年份**：请提供确切的营业执照注册年份（搜索结果不一）。
- [ ] **工厂规模**：厂房面积（平方米）、员工人数、生产线数量。
- [ ] **年产能**：例如 PVC 管材年产多少吨？
- [ ] **ISO 认证**：是否通过 ISO 9001 / 14001？如有，请提供证书编号或扫描件。

## 2. 市场与销售 (Market & Sales)
- [ ] **主要出口市场**：实际上主要卖给哪些国家？（如：东南亚、中东、南美、北美？）
- [ ] **标杆客户**：是否有可以直接列在官网上的知名客户或工程项目名称？

## 3. 产品技术细节 (Product Details)
- [ ] **证书文件**：是否有具体的 UL, CE, 或 ASTM 检测报告？国外客户非常看重这个。
- [ ] **PETG 细节**：PETG 传输管的主要竞争对手是谁？我们的核心优势是价格还是质量？
- [ ] **图片支持**：请提供清晰的产品实拍图、工厂外观图、车间作业图（PDF 无法提取图片）。

## 4. 业务策略 (Strategy)
- [ ] **战略定位**：未来 3 年是想主推高端的 PETG 医疗管，还是继续靠 PVC 电工管跑量？这决定了官网首页的 banner 放什么。
