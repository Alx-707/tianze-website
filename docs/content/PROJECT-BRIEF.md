## 1. 项目概述

### 项目目标

| 优先级 | 目标 | 衡量指标 |
|--------|------|----------|
| **主目标** | 询盘转化 | 询盘表单提交数 |

| 次目标 | 品牌建立 | 回访率、品牌词搜索 |

### 核心差异化

> **Not just a pipe seller — upstream bending machine manufacturer.**

天泽造的是弯管机，然后用自己的机器生产管材。这意味着：
- 深厚的管材成型技术积累
- 快速模具开发能力
- 非标规格灵活定制
- 一线国内品牌 OEM 验证

## 4. 产品体系

### 产品层级

| 层级 | 产品 | 战略定位 | 内容投入 |
|------|------|----------|----------|
| **Tier 1** | PETG 气动传输管 | 差异化、高毛利 | 重点打造 |

| 特殊 | 弯管设备 | 信任背书（不主推销售） | 故事化展示 |

### 产品详情

#### A. 弯管设备（Equipment）

> 战略角色：上游能力，区别于纯贸易商

| ID | 产品 |
|----|------|
| EQ-001 | PVC 半自动弯管机 |
| EQ-002 | PVC 全自动弯管机 |
| EQ-003 | 扩管机 |
| EQ-004 | 切管机 |
| EQ-005 | 定制模具 |

> 应用：建筑电气工程、预埋、穿线

| 材料 | 硬质 PVC |

#### C. PETG 气动传输管系统（Pneumatic Tube）

| 特性 | 高透明、抗冲击、防漏、静音、气密 |

## 5. 目标受众

### 客户画像

| 客户类型 | 关注点 | 对应产品 | 内容策略 |
|----------|--------|----------|----------|
| **承包商/安装公司** | 规范合规、价格、交期 | PVC 电工套管 | 强调 ASTM/Sch80、工程案例 |
| **系统集成商** | 技术规格、定制能力 | PETG 气动管 | 强调透明度、静音、医院应用 |
| **批发商/进口商** | MOQ、供应稳定性 | 全线产品 | 强调产能、合作模式 |
| **OEM 客户** | 模具开发、生产能力 | 定制配件 | 强调设备自研、快速打样 |

### 地理分布

1. 北美（美标 UL651/ASTM）
2. 大洋洲（AS/NZS 标准）
3. 东欧、中东、东南亚
4. 南美、非洲

### 应用场景

#### 场景 A：医院内部物流（PETG 气动管）

| 客户痛点 | 天泽方案 |
|----------|----------|
| 样本运输泄漏 | 防漏设计，生物安全 |
| 不透明管道难排查堵塞 | PETG 高透明，可视化检查 |
| 运行噪音影响环境 | 静音设计，符合医疗标准 |

#### 场景 B：商业/工业电气布线（PVC Sch80）

| 客户痛点 | 天泽方案 |
|----------|----------|
| 预埋时管道被压碎 | Schedule 80 高抗压 |
| 弯头密封差导致进水 | 预制弯头 + 扩口设计 |
| 现场热弯效率低 | 成品弯头（45°/90°） |
| 材料老化 | 耐腐蚀 PVC，30年寿命 |

#### 场景 C：市政/农业给排水（PVC 给水管）

| 客户痛点 | 天泽方案 |
|----------|----------|
| 户外暴晒脆化 | 抗紫外线配方 |
| 化肥/酸碱腐蚀 | 耐腐蚀 PVC |
| 供应商质量不稳定 | ASTM 标准合规 |

## 11. Workflow 链条

```
c-wf (文案生成)
    ↓ 输出: docs/content/{page}/v{N}-final.md
d-wf (设计生成)
    ↓ 输出: docs/design/{page}/HANDOFF.md
i-wf (开发实现)
    ↓ 输出: src/app/[locale]/{page}/
```

**衔接关系**：
- c-wf → d-wf：d-wf 读取 `docs/content/{page}/v{N}-final.md` 作为文案输入
- d-wf → i-wf：i-wf 读取 `docs/design/{page}/HANDOFF.md` 作为开发规范

**c-wf / d-wf 只需读取本文档**，无需读取其他文件。
