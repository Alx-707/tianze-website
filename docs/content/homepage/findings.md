## 品牌核心信息 (from PROJECT-BRIEF)

### 核心差异化
> **Not just a pipe seller — upstream bending machine manufacturer.**

### 三大价值主张
1. **专业电气管道合作伙伴** — ASTM, Schedule 80, 30年寿命
2. **可视化医疗物流专家** — PETG 高透、静音、防漏
3. **工厂直销，灵活定制** — 低 MOQ、快速打样

### 五大竞争优势
1. 系统弯管能力（设备→模具→工艺→配件）
2. 设备研发制造
3. 非标定制 + 快速打样
4. 气动管服务能力
5. 一体化自主生产

### 目标受众
| 类型 | 关注点 |
|------|--------|
| 承包商/安装公司 | 规范合规、价格、交期 |
| 系统集成商 | 技术规格、定制能力 |
| 批发商/进口商 | MOQ、供应稳定性 |
| OEM 客户 | 模具开发、生产能力 |

### 品牌语气
- 专业、可信、技术导向、务实
- 禁用：夸大宣传、未经证实的声明、过度承诺

## 策略发现

### 定位评估结论
- **核心定位有效**："Built by the Engineers Who Build the Machines" 差异化清晰
- **竞争框架明确**：vs 纯贸易商/管材经销商
- **AIDA 结构成熟**：8 区块逻辑完整

### V5 优化方向（13 项微调）

#### 高优先级
1. **Trust Badge** — "15 Years" 与实际运营时间矛盾 → "From Machine Builders to Pipe Makers"
2. **Hero Subheadline** — 公司视角 → 客户收益视角
3. **数据一致性** — 50+ countries 出现4次 → 减少到2次
4. **国家数统一** — PROJECT-BRIEF 说100+，文案说50+ → 统一

#### 中优先级
5. **Section 2 Header** — "No Middlemen. No Guesswork." → "Factory-Direct Precision."
6. **Product Card Tags** — 全部 "In-House Mold" → 差异化标签
7. **Testimonials 归属** — "Project Manager, Middle East" → 增加行业/公司类型
8. **Scenario 6** — "Commercial Building" → "Data Center & Commercial Electrical"

#### 低优先级
9. **Emoji icons** — 换为线条图标 SVG
10. **专利数** — "200+" 需验证，否则改为 "Patented"
11. **Final CTA Headline** — "Ready to Talk?" → "Let's Build Something"
12. **Tertiary CTA** — 邮件订阅与询盘目标冲突 → 删除或降级
13. **Sample CTA Body** — 合并信息，更简洁

## 审查发现

### copy-editing
**审查结论**: 文案整体清晰，语气一致。3 处文字微调，5 处需业务确认。

**必须修改**:
1. Section 2: "tolerances others can't" → "tolerances others can't match"

**建议修改**:
2. Section 3: "Custom OD, wall thickness, and length available" → "...length — all available"
3. Section 6: "Precision you can measure" → "...measure — and verify on arrival"

**需业务确认**:
- 100+ countries 数据来源
- "Patented Bending Technology" 专利文件
- 3-day shipping 承诺
- 24-hour response 承诺
- 6 个 testimonials 真实性

### brand-voice-coach
**审查结论**: 文案符合品牌语气（专业、可信、技术导向、务实）。无夸大宣传。

**合规项 (11项)**:
- 所有核心 headline 和 tagline
- 技术参数声明 (±0.3mm, 16-168mm, Schedule 40/80)
- CTA 语气（务实、低压力）

**需验证项 (7项)**: 与 copy-editing 重叠
- 若无法验证，建议软化：
  - "Patented" → "Proprietary"
  - "Ships within 3 days" → "Typical lead time: 3-5 days"
  - "24-hour response" → "We typically respond within 24 hours"

**不合规项**: 无

### seo-geo
**审查结论**: Meta Tags 8/10, 关键词融入 7/10, GEO 准备度 6/10

**高优先级改进**:
1. Meta Description: 添加 CTA "Get a factory-direct quote"
2. Organization Schema: 确保首页有 JSON-LD
3. Image Alt Text: 确保包含关键词

**中优先级改进**:
4. FAQPage Schema: 添加 2-3 个 FAQ 提升 AI 搜索可见性
5. Hreflang: 确保中英文页面有正确标签
6. Section 3: 考虑自然融入 "PVC electrical conduit"

**GEO 优化建议**:
- 添加 FAQPage Schema (+40% AI 可见性)
- 考虑添加标准引用 "Per ASTM D1785..."

### marketing-psychology (Phase 3b)
**审查结论**: 说服力评分 **8/10** ✅ 通过

**框架一致性**: 10/10
- AIDA 框架完整实现
- 8 区块正确映射到 Attention → Interest → Desire → Trust → Action

**心理学触发点评分**:
| 原则 | 评分 | 说明 |
|------|------|------|
| Authority Bias | 9/10 | "From Machine Builders" 建立技术权威 |
| Social Proof | 8/10 | 证言归属优化，100+ countries 数据 |
| Commitment (Foot-in-Door) | 9/10 | 免费样品低门槛 |
| Specificity | 9/10 | ±0.3mm, 16-168mm 具体数字 |
| Reciprocity | 8/10 | 先给价值再要承诺 |
| Loss Aversion | 7/10 | "Test Before You Commit" 隐含风险规避 |

| Scarcity/Urgency | 4/10 | **缺失** — 无紧迫感触发 |

**可选优化（非阻断）**:
1. Final CTA 添加风险逆转徽章："No Obligation · Free Consultation"
2. Section 4 考虑添加稀缺性提示
3. Hero 考虑添加动态社会证明

**结论**: V5 通过 Deep 模式说服力审查，无需触发优化轮。
