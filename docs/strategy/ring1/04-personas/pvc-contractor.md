# Persona: PVC Contractor

> Business line: Pipes (PVC Conduit)

## Profile

**Name archetype:** James — Electrical Contractor / Project Engineer
**Company:** Electrical contracting firm handling commercial/industrial projects
**Region:** North America, Australia
**Company size:** 20-200 employees
**Role:** Project Engineer or Procurement Specialist

## Job to Be Done

"Source PVC conduit fittings that meet project specifications exactly — right standard, right schedule, right dimensions — delivered on time so my project doesn't get delayed."

## Decision Chain

1. **Trigger:** New project requires specific conduit specs (Schedule 40/80, specific bend angles)
2. **Search:** Check existing supplier catalog, then Google for specific specs if gaps exist
3. **Shortlist:** Based on spec match, certification, and lead time
4. **Evaluate:** Verify technical compliance, request spec sheets, confirm lead time
5. **Decide:** Purchase order based on project timeline — speed matters more than price
6. **Others involved:** Project manager (timeline), site supervisor (installation compatibility)

## Information Needs by Stage

| Stage | Needs | Content that answers |
|-------|-------|---------------------|
| Awareness | Does this manufacturer make the exact spec I need? | Product pages with detailed specs |
| Comparison | How do tolerances compare? Is it ASTM/UL compliant? | Spec sheets, standards comparison tables |
| Decision | Can they deliver within my project timeline? | Lead time info, shipping options, contact for urgent quotes |

## Top 3 Concerns

1. **Spec compliance** — "Will this pass inspection on my job site?"
2. **Lead time** — "I need it in 3 weeks, not 3 months"
3. **Dimension accuracy** — "plus/minus tolerances matter for installation fit"

## How They Describe What They Want

- "I need Schedule 80 90-degree bends, 2-inch, ASTM D2467"
- "What's your shortest lead time for a full container?"
- "Do you have test reports?"
- "Can you match this drawing?"
