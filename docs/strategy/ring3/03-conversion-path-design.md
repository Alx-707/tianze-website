# 转化路径设计

> Ring 3 | 已确认（2026-03-30）

## 一句话结论

买家到站后，应该在 **2 次点击内** 走到 inquiry。  
路径要短，CTA 要分层，不要让高意向买家在站里迷路。

## 当前 CTA 分层

### 低承诺

- 查看认证状态
- 下载规格表

适合还在比较的人。

### 中承诺

- 申请免费样品

适合愿意先试，但还没准备直接下询盘的人。

### 高承诺

- 获取报价
- 预约技术沟通
- 提交定制需求

适合已经接近决策的人。

## 当前页面部署规则

- 首页：主打分流 + 样品 / 报价入口
- 产品页：规格下载 + 样品 + 报价
- 设备页：技术咨询优先
- 定制页：提交需求优先
- Contact：最终统一转化页

## 当前转化原则

- 不同业务线 CTA 语气可以不同，但入口逻辑要统一
- 高意向路径不能藏在深层页面
- 每个页面都要有清楚的下一步，而不是“看完就结束”
