## Core Copy Principles

### 1. Verifiable claims replace self-assessment

| Never write | Write instead |
|-------------|--------------|
| "Industry-leading quality" | "±0.3mm tolerance on every bend" |
| "Reliable supplier" | "98% on-time delivery across 20+ countries" |
| "Advanced technology" | "We design the bending machines that form every fitting" |
| "Excellent service" | "Email response within 24 hours" |

### 2. Results before process

Lead with what the buyer gets, then explain how.

> "Tighter tolerances on every bend — because we engineer the machines that form them."
> NOT: "We have advanced bending machines which allow us to achieve..."

### 3. Buyer language, not seller language

Use words from buyer question chains (Task 6), not marketing jargon.

| Seller language | Buyer language |
|----------------|---------------|
| "Comprehensive product portfolio" | "Schedule 40 and 80, full range 1/2" to 6"" |
| "Turnkey solutions" | "From drawing to sample in 15 days" |
| "Value-added services" | "Free samples, we pay return freight if unsatisfied" |

### 4. Causal claims over bare assertions

The precision narrative only works if we explain WHY:

```
[Benefit] because [Reason backed by fact]

"Precision you can measure — because we design the bending machines ourselves."
"Consistent quality batch after batch — because we control the entire chain from machine to product."
"Non-standard angles available — because we build the machines, we're not limited by someone else's tooling."
```

## Per-Business-Line Copy Tone

### Pipes (PVC Conduit + PETG)

**Tone**: Technical authority + factory confidence
**Register**: Professional but not academic. Engineer talking to procurement.

Key copy patterns:
- Lead with standards compliance: "Meets AS/NZS 2053 requirements"
- Follow with measurable specs: "Wall thickness ±0.15mm"
- Close with action: "Download spec sheet" or "Request free samples"

**PVC-specific vocabulary**: Schedule 40/80, bell end, socket, conduit body, sweep bend, ASTM D2467, UL 651
**PETG-specific vocabulary**: transparency grade, burst pressure, pneumatic delivery, seal integrity, hospital-grade

### Equipment (Bending Machines)

**Tone**: Engineering expertise + practical confidence
**Register**: Technical depth, but accessible. Factory owner talking to factory owner.

Key copy patterns:
- Lead with buyer's problem: "Your customers are asking for bends. You're losing orders."
- Present the solution as ROI: "Add bending capability. Payback in [X] months."
- Prove with self-validation: "We use these machines in our own production every day."

**Vocabulary**: diameter range, production rate (pcs/hour), infrared ceramic heating, hydraulic pressure, hot-bending vs injection molding

**Important**: Iterative evolution narrative, NOT competitor comparison. Show Gen 1→2→3 improvements. "Our third-generation machine processes 40% faster with 25% less energy."

### Custom Molds / OEM

**Tone**: Partnership-oriented + capability showcase
**Register**: Reassuring. Senior engineer talking to product manager.

Key copy patterns:
- Lead with capability: "Send us your drawing. We'll develop the mold."
- Address flexibility: "Low MOQ for trial runs. Scale when you're ready."
- Close with low-risk entry: "Start with 5,000 pieces."

**Vocabulary**: mold development, CAD-to-sample, private label, production run, tooling

## Copy Don'ts (Anti-Patterns)

| Don't | Why |
|-------|-----|
| "World-class" / "Top-tier" / "Best-in-class" | Unverifiable self-assessment |
| "One-stop solution" / "Turnkey" | Generic marketing jargon |
| "Dear valued customer" | Formal and impersonal |
| "We are committed to..." | Empty promise language |
| Competitor names or direct comparison | Owner preference: iterative self-improvement, not benchmarking |
| "Cheap" or "lowest price" | Use "factory-direct" or "competitive pricing" |
| Long paragraphs without data | Every claim needs a number or evidence |
