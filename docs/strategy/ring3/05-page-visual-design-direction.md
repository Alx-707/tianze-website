# 页面视觉方向

> Ring 3 | 已确认（2026-03-30）

## 一句话结论

视觉要传达的是：

> **工业制造的精度感 + 现代数字产品的克制感**

不是：

- Alibaba 展示页
- 泛 SaaS 模板
- 花哨炫技 demo

## 当前视觉原则

### 1. 工程精度要成为视觉语言

适合放大的内容包括：

- 规格表
- 公差
- 标准对照
- 工艺流程
- 设备演进

### 2. 先展示证据，再做装饰

视觉元素首先要服务：

- 制造能力
- 产品理解
- 信任建立

不是为了让页面显得“设计感很强”。

### 3. 信息密度可以高，但必须可扫读

买家不会像看品牌故事一样慢慢读。  
页面应该更像：

- 好扫的结构化信息页
- 带审美控制的工程页面

## 当前页面方向

- 首页：分流 + 差异化 + 信任
- 产品页：规格与标准优先
- 设备页：机器、能力、产能优先
- About：工厂、流程、信任优先
- Contact：清楚、直接、低干扰
