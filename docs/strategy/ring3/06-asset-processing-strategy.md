# 资产处理策略

> Ring 3 | 已确认（2026-03-30）

## 一句话结论

天泽现在不是没有素材，  
而是素材还没被处理成真正可用的站点资产。

## 当前原始资产类型

- 工厂外景 / 车间图
- PVC 产品图
- PETG 产品图
- 设备图
- 规格表 / 数据表
- 认证文件
- 工厂 / 生产视频

## 当前处理原则

### 产品图

- 背景干净
- 构图统一
- 能看清尺寸和结构
- 不是简单把产品抠出来就完

### 设备图

- 要有工厂场景图
- 也要有相对干净的展示图
- 最好能看到设备正在工作

### 工厂图

- 真实
- 整洁
- 有制造感
- 不能像库存图

### 认证文件

- 扫描清楚
- 展示统一
- 下载版本和页面展示版本分开处理

## 当前边界

- 不做过度修饰
- 不把素材修成失真宣传图
- 优先追求“真实但专业”
