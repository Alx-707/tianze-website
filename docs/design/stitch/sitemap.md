## 网站概览

**网站目标**: 询盘转化 + 产品展示（面向海外买家选择供应商）

**主要转化路径**:
```
首页(建立信任) → 产品(了解规格) → 联系(获取报价)
         ↘ 关于(深入了解) ↗
```

## 响应式策略

**设计优先级**: Desktop-first（B2B 用户主要用桌面端）

**断点定义**:

| 断点 | 宽度 | 适配重点 |
|------|------|----------|
| Mobile | < 768px | 简化导航，垂直堆叠，保留核心 CTA |
| Tablet | 768px - 1024px | 调整网格，保持可读性 |
| Desktop | > 1024px | 完整体验，多列布局 |

**各页面适配要求**:

| 页面 | Mobile | Desktop |
|------|--------|---------|
| 首页 Hero | 居中堆叠，图片缩小 | 可左右分栏 |
| 产品列表 | 单列卡片 | 3-4 列网格 |
| 产品详情 | 信息垂直排列 | 左右分栏（图片+详情） |
| 联系表单 | 全宽表单 | 左右分栏（表单+信息） |
