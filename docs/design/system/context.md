## 目标用户

### 主要用户：海外分销商/进口商

**画像**:
- 地区: 东欧、北美、南美、中东、大洋洲、非洲、东南亚
- 角色: 建材批发商、进口商
- 规模: 中小型

**核心需求**:

| 需求 | 优先级 | 我们如何满足 |
|------|--------|--------------|
| 产品质量稳定 | 高 | ISO 9001:2015 认证，100% 新料 |
| 价格有竞争力 | 高 | 工厂直供，无中间商 |
| 交期可靠 | 高 | 全链路自主生产 |
| 小批量试单 | 中 | 低 MOQ 支持 |
| 定制规格 | 中 | 快速模具开发能力 |

**决策旅程**:
1. 认知: Google 搜索 / 阿里巴巴 / 展会
2. 考虑: 对比 3-5 家供应商（规格、价格、认证）
3. 决策: 样品测试 → 小批量试单 → 长期合作

**访问网站时的关键问题**:
- 你们是工厂还是贸易商？
- 有哪些认证（UL/ASTM/AS/NZS）？
- MOQ 是多少？
- 能否定制非标规格？
- 如何联系/获取报价？

### 次要用户：专业系统集成商

**画像**: 医院/实验室物流系统集成商

**核心需求**:
- PETG 管道透明度、耐冲击
- 定制弯头和连接件
- 稳定供货能力

### 次要用户：工程承包商

**画像**: 建筑电气工程承包商

**核心需求**:
- 符合图纸规格（ASTM/Sch80）
- 安装便捷（预制弯头）
- 快速交付

## 避免事项 (Anti-patterns)

| 避免 | 原因 |
|------|------|
| ❌ 大量产品堆砌、阿里巴巴风格 | 看起来像廉价贸易商 |
| ❌ 过度 SaaS 模板感（过于圆润、渐变过多） | 与工业制造不匹配 |
| ❌ 暗色+橙色老套工业配色 | 缺乏现代感 |
| ❌ 通用库存图片 | 无法体现真实工厂能力 |
| ❌ 过多装饰性动画 | 分散注意力，降低专业感 |
