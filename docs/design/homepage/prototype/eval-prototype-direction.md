## 逐原型分析

### v1 "Navy+Amber"

**一句话**：稳妥的工业 B2B 方案 — 专业但缺少记忆点。

**视觉方向**：深海军蓝 + 琥珀色强调，IBM Plex Sans + JetBrains Mono 等宽字体。微小圆角（2-4px），网格纹理暗示精密测量。整体走 Vercel 式克制路线。

**优势**：
- 配色传递可靠感，蓝色系天然适合 B2B 信任建立
- 等宽字体用于规格参数（`±0.3mm`），强化技术感
- 网格背景纹理呼应工程制图

**劣势**：
- 与大量 B2B SaaS 站点视觉重叠，缺乏行业辨识度
- Amber 作为强调色力度偏弱，不够果断
- 整体偏「安全」，没有传达出「我们造设备」的硬核气质

---

### v2 "Twitter Blue"

**一句话**：SaaS 产品页误入制造业 — 调性完全错位。

**视觉方向**：Twitter/X 蓝（#1d9bf0）为唯一强调色，Outfit 圆体字，全圆角按钮（pill shape），大圆角卡片（16-20px）。极简扁平，轻阴影。

**优势**：
- 排版精细度高，间距系统严谨（8px grid）
- 视觉执行质量好，Outfit 字体现代感强
- 交互细节考究（hover 上浮 + 蓝色光晕）

**劣势**：
- 看起来像 Stripe、Linear、Vercel — 不像制造商
- 全圆角 pill 按钮和蓝色光晕完全是科技消费品语言
- 采购经理会觉得这是个代理商或 SaaS 平台，不是工厂
- 蓝色系在 B2B 制造业中毫无差异化

---

### v3 "The Forge"

**一句话**：锻造厂美学 — 唯一让人觉得「这家公司弄脏过手」的方案。

**视觉方向**：纯墨黑 + 单一红色强调（#D63031），Archivo Black 粗体展示字 + Manrope 正文，零圆角，网格纸纹理，顶部 3px 红色品牌条，尺规分割线（ruler）带测量刻度。

**优势**：
- 零圆角 + 粗体大写标题 = 机械切割感，与管件制造高度吻合
- 红色品牌条 + 红色左侧边线 = 简洁但有力的品牌识别系统
- Archivo Black 超粗体 + 全大写 = 工业力量感，与众不同
- 产品卡 hover 时底部红线滑入，像金属被加热 — 动效语义一致
- `[01]` `[02]` 方括号编号 = 工程图纸编号系统，细节到位
- 尺规分割线（ruler with measurement marks）— 独创，行业内从未见过

**劣势**：
- 纯黑+红有潜在的「过于激进」风险，部分保守市场（日本、北欧）可能感觉压迫
- 暖白背景（#F5F3F0 paper）缓解了这个问题，但仍需注意深色区块的比例控制
- Archivo Black 字号如果控制不当，在小屏上可能显得粗暴

---

### Stitch A "Clean Precision"

**一句话**：高完成度的安全方案 — 像模板，但确实好看。

**视觉方向**：白底为主，蓝色强调（与 v2 类似但更克制），产品图使用真实照片（蜂巢 PVC 管截面等），布局清晰工整。有实际照片（工厂航拍、弯管设备），完成度远高于 HTML 原型。

**优势**：
- 图片质量高，产品展示专业，蜂巢截面图有视觉吸引力
- 布局层次清晰，信息密度适中
- 有导航栏、完整页脚 — 最接近成品的原型

**劣势**：
- 与 Alibaba 金牌供应商页面、GlobalSources 供应商首页高度相似
- 蓝色强调 + 白底 + 卡片式产品展示 = B2B 制造业「标准模板」
- 没有任何视觉元素传递「我们自研弯管设备」的核心差异化
- 虽然精致，但采购经理看到后不会记住这个品牌

---

### Stitch B "Dark Forge"

**一句话**：v3 精神的 Stitch 演绎 — 暗色调工业感，但执行略粗糙。

**视觉方向**：深色为主（接近 v3 的墨黑），红色强调色，粗体斜体展示字，产品使用黑白/低饱和度照片，工业场景（数据中心、基建）占比大。暗色调 + 红色品牌条。

**优势**：
- 暗色调在制造业网站中确实差异化明显
- 红色强调 + 暗底的对比度很好
- 应用场景区块的照片（数据中心、商业楼宇）增强了专业感
- 黑白产品图 + 红色标签 = 有设计意图

**劣势**：
- 斜体展示字（"Machine Makers"）的字体选择不如 v3 的 Archivo Black 有力
- 暗色区块占比过高，整页偏压抑，缺少 v3 那种明暗交替的节奏
- 某些区块（产品线）在暗背景上信息可读性打折
- 红色强调的使用不如 v3 克制 — v3 的红色是精确的手术刀，Stitch B 更像大面积涂抹

| 排名 | 原型 | 总分 | 理由 |
|:---:|------|:---:|------|
| 1 | **v3 "The Forge"** | 43 | 唯一建立了完整且独特的设计语言系统。零圆角、尺规线、方括号编号、红色品牌条 — 每个细节都在说「精密制造」。在 B2B 制造业网站中辨识度最高。 |
| 2 | **v1 "Navy+Amber"** | 38 | 稳妥的备选方案。如果客户认为 v3 过于激进，这是最安全的退路。专业可靠，但没有差异化记忆点。 |
| 3 | **Stitch B "Dark Forge"** | 37 | 方向正确（暗色+红色），但执行不如 v3 精准。可作为 v3 视觉方向的参考补充。 |

## 最终建议

**推荐 v3 "The Forge" 作为视觉方向基底。**

核心原因：

1. **品牌真实性** — Tianze 的核心差异化是「造设备的人在造管道」。v3 是唯一一个在视觉层面让人感受到「这家公司弄脏过手」的方案。零圆角、粗体大写、尺规分割线、方括号编号 — 每个设计决策都在强化「精密制造」的叙事。

2. **差异化** — 在 B2B 制造业网站中（充斥着蓝色+白底+圆角卡片），v3 的视觉语言是独一无二的。采购经理在比较 5 家供应商时，v3 是唯一会被记住的。

3. **系统完整性** — 不是某个元素好看，而是所有元素形成了一个自洽的系统：颜色（单色+红）、字体（Archivo Black + Manrope + JetBrains Mono 三层级）、圆角（零）、分割线（尺规）、编号（方括号）、动效（底部红线滑入）。这意味着它可以可靠地扩展到产品页、关于页、联系页，而不会在扩展过程中失去一致性。

**实施建议**：
- 从 Stitch A/B 借鉴图片处理方式（产品照片、工厂航拍），嫁接到 v3 的设计系统中
- 注意暗色区块与亮色区块的比例（建议 40:60 亮底为主），避免整页压抑
- 考虑在保守市场（日本站）准备 v1 的备用配色方案
