# Research Workspace

这个目录现在只保留**还和当前 Tianze 主线有实际关系**的研究材料。

当前保留重点：

- 竞品综合分析
- 市场研究
- 产品目录结构研究
- 设计输入来源分层

已经移出的内容：

- 偏工具研究
- 偏元工作流研究
- 和当前主线不贴的技术调研
- VPS 托管迁移这类当前不在主线上的资料

使用规则：

- 这里是输入层，不是规则层
- 如果某份研究没有被当前设计、内容、产品结构或治理决策继续使用，就不应该长期挂在主树
- 当前规则与真相仍然以 `docs/guides/` 为准

当前设计前置整理：

- `design-input-source-map.md`：把现有 docs 分成「当前可用真相 / 可用但需清洗 / 历史参考」三类，避免后续设计工作把旧材料和当前真相混用。
- `tianze-internal-evidence-inventory.md`：盘点 Tianze 目前能证明什么、哪些证据缺失、哪些公开 claim 需要谨慎处理。
