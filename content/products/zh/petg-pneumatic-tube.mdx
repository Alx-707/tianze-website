# PETG气动物流管系统

优质PETG（聚对苯二甲酸乙二醇酯）透明管材，专为气动输送系统设计。产品具有卓越的透明度、耐久性和气密性能，满足高要求的物流应用。

## 核心特点

- **晶莹透明**：可视化监控载体运行状态
- **静音运行**：光滑内壁降低输送噪音
- **密封可靠**：精密加工接口确保气密性
- **抗冲击**：PETG材质承受载体碰撞
- **耐化学**：适用于医药环境

## 技术参数

| 参数 | 规格 |
|------|------|
| 材质 | PETG（可选食品级） |
| 外径 | 110mm、160mm |
| 壁厚 | 2.3mm、3.0mm |
| 弯曲半径 | R650、R800、R1000 |
| 最大工作压力 | 0.5 MPa |
| 温度范围 | -20°C 至 +60°C |

## 系统配件

- **直管**：标准长度3m、6m，可定制
- **弯管**：45°、90°多种半径
- **连接器**：推入式和法兰式
- **分流器**：Y型分岔路由
- **检修口**：维护面板

## 应用领域

- **医院药房**：药品配送至病房
- **实验室网络**：样本输送系统
- **银行网点**：单据及现金处理
- **工业设施**：小件物料物流
- **零售仓库**：订单履行系统

- 尺寸检验
