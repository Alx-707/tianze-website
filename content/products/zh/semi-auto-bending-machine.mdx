# 半自动PVC弯管机

专业级弯管设备，专为PVC电工套管制造设计。我们的半自动机型在精密控制与操作灵活性之间实现完美平衡。

## 核心特点

- **精密温控**：数字温度管理系统确保弯曲质量稳定一致
- **角度可调**：支持45°、90°及自定义角度弯曲
- **多规格兼容**：适配DN20-DN110mm直径管材
- **快换模具**：快速换模系统适应不同规格生产
- **安全保障**：配备急停开关、过热保护及安全防护罩

## 技术参数

| 参数 | 规格 |
|------|------|
| 管材直径 | DN20-DN110mm |
| 弯曲角度 | 45°、90°、自定义 |
| 加热方式 | 红外线/电热 |
